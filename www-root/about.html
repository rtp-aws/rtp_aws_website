About

rtp-aws.org is a user group started in 2020. Each week we have a one hour workshop. During these workshops we:

  • sometimes just chat 8-)
  • work on projects
  • work on free labs provided by AWS

This website is hosted on AWS, as a static website using S3. It uses CloudFront for CDN and Route53 for DNS.
